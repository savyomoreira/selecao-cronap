# selecao-cronap
Código desenvolvido no processo de seleção da Cronapp

tecnologias ultilizadas:
  Backend:
    Java 8;
    Spring(Boot, Data, Rest);
    JPA(Hibernate);
    Thymeleaf;
    
  Frontend:
    AngularJS;
    Bootstrap;
    
  Banco de Dados:
    H2
    
        
OBS: a aplicação responde a requisiçoes feitas a porta 8080.: ex http://localhost:8080/
